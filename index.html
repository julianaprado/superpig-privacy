Tipos de Dados Coletados

Entre os tipos de Dados Pessoais que esta Aplicação coleta, por si própria ou através de terceiros, estão: Rastreadores; Dados de Uso; endereço de e-mail; nome de usuário; ID do Usuário; endereço IP; informações do dispositivo; informações do navegador; sistemas operacionais; cidade; CEP; estado; região; país; localização precisa; localização aproximada; primeiro nome; último nome; senha; foto; permissões de dispositivo; UUID; dados de crash.

Detalhes completos sobre cada tipo de Dados Pessoais coletados são fornecidos nas seções dedicadas desta política de privacidade. Os Dados Pessoais podem ser livremente fornecidos pelo Usuário, ou, no caso de Dados de Uso, coletados automaticamente ao usar esta Aplicação.

A menos que especificado de outra forma, todos os Dados solicitados por esta Aplicação são obrigatórios e a falha em fornecer estes Dados pode tornar impossível para esta Aplicação fornecer seus serviços. Os Usuários são responsáveis por quaisquer Dados Pessoais de terceiros obtidos, publicados ou compartilhados através desta Aplicação.

Modo e Local de Processamento dos Dados

Métodos de Processamento

O Proprietário toma medidas de segurança apropriadas para prevenir acesso não autorizado, divulgação, modificação ou destruição não autorizada dos Dados. O processamento de Dados é realizado usando computadores e/ou ferramentas habilitadas para TI, seguindo procedimentos organizacionais estritamente relacionados aos propósitos indicados.

Local

Os Dados são processados nos escritórios operacionais do Proprietário e em outros locais onde as partes envolvidas no processamento estejam localizadas. Dependendo da localização do Usuário, transferências de dados podem envolver transferir os Dados do Usuário para um país diferente do seu.

Tempo de Retenção

A menos que especificado de outra forma, os Dados Pessoais devem ser processados e armazenados pelo tempo necessário para o propósito para o qual foram coletados e podem ser retidos por mais tempo devido a obrigação legal aplicável ou baseado no consentimento dos Usuários.

Os Propósitos do Processamento

Os Dados relativos ao Usuário são coletados para permitir ao Proprietário fornecer seu Serviço, cumprir suas obrigações legais, responder a solicitações de execução, proteger seus direitos e interesses, detectar qualquer atividade maliciosa ou fraudulenta, bem como: Análises, Comentários de conteúdo, Exibição de conteúdo de plataformas externas, Processamento de pagamentos, Registro e autenticação, Gerenciamento de contatos e envio de mensagens, Hospedagem e infraestrutura de backend, Interações baseadas em localização, Recursos sociais, Permissões do dispositivo para acesso a Dados Pessoais e Monitoramento de infraestrutura.

Base Legal do Processamento

O Proprietário pode processar Dados Pessoais se:

  • Os Usuários deram seu consentimento
  • É necessário para execução de um contrato
  • É necessário para cumprimento de obrigação legal
  • É necessário para interesses legítimos do Proprietário

Os Direitos dos Usuários

Os Usuários podem exercer certos direitos sobre seus Dados:

  • Retirar consentimento a qualquer momento
  • Objetar ao processamento de seus Dados
  • Acessar seus Dados
  • Verificar e buscar retificação
  • Restringir o processamento
  • Ter Dados excluídos
  • Portabilidade de dados
  • Registrar reclamação

Limitação de Responsabilidade

Alterações nesta Política

O Proprietário reserva-se o direito de alterar esta política a qualquer momento, informando adequadamente os Usuários. O uso contínuo do Serviço significará a aceitação da política revisada. Se os Usuários não concordarem com as mudanças, devem parar de usar o Serviço.

Esta política é regida pela lei brasileira. Para Usuários que se qualificam como Consumidores no Brasil, a lei brasileira se aplicará.

Para Usuários no Brasil que se qualificam como Consumidores, as disputas serão resolvidas conforme o Código de Defesa do Consumidor e legislação aplicável.
